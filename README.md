# ethanol_controller
This repository contains the Ethanol controller in Python 2. 7/POX


# Installation #

Clone the repository and update submodules
```bash
git clone https://github.com/h3dema/ethanol_devel.git
cd ethanol_devel
git submodule update --remote
```
